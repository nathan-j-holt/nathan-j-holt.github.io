Pizza Builder

This tool was developed to gain experience with MSSQL by allowing users to select pizza items, which are then stored in a database. It also includes a rewards system that enables users to accumulate points and redeem them for discounts on future pizza purchases when they have enough points. All relevant information, including rewards data, is stored in the database.
